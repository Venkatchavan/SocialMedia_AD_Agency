# PLAYBOOK.md

## Pipeline: TikTok + Meta → Creative Intel → Brief

### Stage 1 — Plan
- workspace_id
- platforms
- date range
- max assets per brand
- competitor set
- output format

### Stage 2 — Collect
TikTok:
- search keyword/date/count
- store: caption, thumbnail, public metrics, public comments (anonymized later)

Meta:
- import Ad Library URL per brand
- store: creative metadata, impression range (if available), copy/headline/CTA

### Stage 3 — Analyze Media
Return deterministic tags:
- asset_type, format_type
- hook_tactics
- messaging_angle
- offer type/terms
- proof elements
- CTA type
- funnel stage
- risk flags

### Stage 4 — Comment Mining
Output anonymized:
- FAQs
- objections
- desire language (rewritten)
- confusion points
- angle suggestions

### Stage 5 — Synthesize
- cluster “winners” (distribution proxy + recency)
- extract repeatable patterns
- produce testing matrix

### Stage 6 — Brief
Use BrandBible + BriefTemplate:
- SMP + RTBs + Mandatories
- directions (2–4)
- hook bank (10–20)
- scripts (3–6)
- shotlists + on-screen text
- test matrix

### Stage 7 — QA Gate
- PII scan
- no-copy check
- claim risk checks
- uncertainty labeling

## Code Size Rule
All implementations must follow: **no single code file > 250 lines.**
