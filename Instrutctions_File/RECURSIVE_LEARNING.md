# RECURSIVE_LEARNING.md

## Goal
Improve prompts/playbooks weekly without destabilizing the system.

## Weekly Loop
1) Select wins/fails (top 10 each)
2) Compare tags, hooks, offers, proofs
3) Change ONE thing (prompt OR taxonomy OR collector)
4) Run regression on fixed benchmark assets
5) Version + changelog + rollback

## Artifacts
- prompts/vN/
- taxonomy/vN/
- playbooks/vN/
- benchmarks/
- changelog.md
